Penginstalan local :

Aplikasi Ini dibangun menggunakan
1. Front-end : NEXT JS
2. Back-end : NODE JS
3. CSS Framework : TailwindCSS

Prasyarat Komputer: 
1. Terinstall NPM
2. Terinstall Javascrip
3. Terinstall Node JS

Langkah 1 jalankan aplikasi di server lokal dengan terminal diletakkan di direktori folder gpacheby_imk dengan perintah 
npm run dev
# or
yarn dev
# or
pnpm dev

Langkah 2 buka aplikasi di peramban browser seperti google chrome atau safari
Open http://localhost:3000 with your browser to see the result.
